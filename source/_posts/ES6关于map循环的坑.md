### 关于map循环+使用模板字符串

​	```

​	 var arr = ['张三','李四','玩物'];

        var html ="";

​        arr.map(item=>{

            html+=`<li>

                ${arr.map(value=>{

                    return `<span>${value}</span>`

                })}

            </li>`

        })

​        $("#test").html(html);

​	```
